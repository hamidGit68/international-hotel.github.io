この度は、COM4t (Font Factory Field）のフリーフォントをダウンロードしていただき、誠にありがとうございます。沢山の皆様に喜んでもらえる事はもちろんの事、デザインのプロの方々にも喜んでもらえるような質の高いフォントを作成し、精進していく所存でございます。まだまだ若輩者ですがこれからも宜しくお願いします。

当フォントは、個人・商用問わず、遠慮なくどこでも使って頂いて結構です。加工するなり好きなように使って下さい。しかし、著作権は放棄していないので、当フォントデータを販売または再配付等はしないでください。雑誌・メディア等に転載して頂ける場合も御連絡していただかなくても結構です。もし、わからない点や、ご質問等がございましたら、サイト内（http://com4t-fff.seesaa.net/）に質問コーナーを設けてありますので、そちらに書き込んでいただければと思います。

H e l p

Macintosh版には、Postscript type1 font(P.S)とTrue Type Font(T.T)の両方がありますが、あなたのOS環境にあわせて、どちらかをお使い下さい。あなたのハードにAdobe Type Managerが入っていればP.Sの方で。なければT.Tの方をインストールして下さい。もし、どちらをインストールして良いか解らない場合は、T.Tの方を入れて置けば問題はないと思います。
